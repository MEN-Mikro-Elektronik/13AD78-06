PU5 specific Macros

Notes about the values in the macros below

The conversion between raw ADC values read from firmware and physical values (mA, mV) must be done such that no floating point arithmetic is necessary.

Origin of values below: 

  3.3V measured via divider   1k /1k  (x2) => 3000mV x   2 = 6000
    5V measured via divider   2k7/1k  (x3.7) => 3000mV x 3.7 = 11100
   12V measured via divider  10k /2k7 (x4.7) => 3000mV x 4.7 = 14100

    I=0..20A
