Detailed Description

Windows service for the AD78 driver.

Author:
dieter.pfeuffer@men.de
Date
2009/08/31 14:35:44
Revision
1.2

The service initiates a PSU power off (AD78_SWOFF) during Windows shutdown and performs a Windows shutdown if requested from the PSU (polls AD78_DOWN_EVT).

Required: libraries: mdis_api, usr_oss

Preprocessor Switches:
DBG
